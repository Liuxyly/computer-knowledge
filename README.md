# computer-knowledge 计算机知识库

> 持续更新的计算机知识笔记，聚焦 Web 开发与 Spring Boot 生态

## 📚 内容概览

本仓库专注于 **Web 开发** 方向，以 Spring Boot 为核心，逐步构建完整的知识体系。

### 🗂️ 知识目录

```
computer-knowledge/
├── web-basics/           # Web 基础
│   ├── http/            # HTTP 协议
│   ├── rest-api/        # RESTful API 设计
│   └── security/        # Web 安全
├── spring-boot/         # Spring Boot 核心
│   ├── core/            # 核心概念
│   ├── data/            # 数据访问 (JPA, MyBatis)
│   ├── web/              # Web MVC
│   └── validation/      # 参数校验
├── spring-ecosystem/    # Spring 生态
│   ├── spring-cloud/    # 微服务
│   ├── cache/           # 缓存 (Redis, Caffeine)
│   ├── message/         # 消息队列 (RabbitMQ, Kafka)
│   └── schedule/        # 定时任务
├── database/            # 数据库
│   ├── mysql/
│   └── redis/
├── architecture/       # 架构设计
│   ├── design-patterns/# 设计模式
│   └── clean-code/     # 代码整洁之道
└── algorithms/         # 数据结构与算法
```

## 🎯 学习路径

### 入门 (1-2周)
1. HTTP 基础 + RESTful API 规范
2. Spring Boot 入门与核心注解
3. Spring Data JPA / MyBatis 基础

### 进阶 (2-4周)
1. Spring MVC 深入 + 拦截器/过滤器
2. Spring Boot 异常处理与参数校验
3. Redis 缓存设计与应用
4. 数据库事务与锁

### 高级 (4周+)
1. Spring Cloud 微服务基础
2. 消息队列 (RabbitMQ/Kafka)
3. 分布式系统设计
4. 高并发实战

## 📖 笔记规范

每个知识点采用 **黄金圈** 模板：

```markdown
## 主题名称

### 是什么 (What)
> 核心概念，一句话说明白

### 为什么 (Why)
> 解决的问题/优势/使用场景

### 怎么用 (How)
> 代码示例 + 注意事项
```
